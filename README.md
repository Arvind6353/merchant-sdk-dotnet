This repository contains the PayPal Merchant SDK C#.NET Class Library Application and the PayPalAPISample Sample ASP.NET C# Web Application.


SDK Integration
---------------
*	Integrate the PayPal Merchant SDK with an ASP.NET Web Application

*	Use NuGet to install the 'PayPal_Merchant_SDK' package 

*	The NuGet package installs the dependencies to the solution and automatically updates the project

*	Dependent library references:
	•	'PayPal_Merchant_SDK.dll'
	•	'PayPal_Core_SDK.dll'	
	•	'log4net.dll'

*	Namespaces:
	•	PayPal
	•	PayPal.PayPalAPIInterfaceService
	•	PayPal.PayPalAPIInterfaceService.Model
	•	PayPal.Util
	•	PayPal.Exception
